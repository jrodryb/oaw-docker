Servicio de diagnóstico

El servicio on-line de diagnóstico de la Accesibilidad web remitirá en diferido, al correo electrónico indicado, el informe de la web o código fuente analizado. Las peticiones son encoladas y procesadas progresivamente.

Puede realizar el análisis de una página web pública en Internet, o del código fuente de una página interna a la que el analizador no pueda acceder (por ejemplo una Intranet o una página que requiera autenticación previa mediante certificado electrónico).

Nota: En caso de que rellene los campos URL y código, se analizará unicamente la URL.

Si tiene cualquier problema con el servicio, por favor, envíe un correo electrónico al Observatorio de Accesibilidad Web .

Destinatario
Seleccionar el tipo de análisis

  • Analiza un sitio web a partir de una URL. Análisis online de la/s página/s de un sitio web público en Internet a partir de una URL inicial.

  • Analiza un conjunto cerrado de URLs. Análisis online de un conjunto predefinido de páginas de un sitio web público en Internet.

  • Analiza el código fuente introducido Análisis del código fuente de una página (p. ej. página de una Intranet a la que el analizador no pueda acceder).

  • Mixto: Analiza el código fuente y las urls introducidas Análisis del código fuente de una página (p. ej. página de una Intranet a la que el analizador no pueda acceder).

Opciones de rastreo

Si desea que la selección aleatoria de páginas se realice dentro del último "directorio" (hasta la última "/") indicado en la URL (por ejemplo http://www.dominio.com/directorio/pagina), marque la siguiente opción.

Análisis de conjunto de URLs

Introduce una URL por línea. Solo se analizarán un máximo de 51 páginas

Análisis de código fuente

NOTA: No se analizará el código HTML generado con JavaScript.

Tipo de informe

El servicio generará un informe con el detalle de todos los problemas según la metodología de alguno de los siguientes Observatorios de Accesibilidad:

  • Informe conforme a las comprobaciones 2.3 de la metodología del Observatorio de Accesibilidad, basada en la UNE 139803:2012.
  • Informe conforme a la metodología del Observatorio de Accesibilidad, basada en la UNE-EN 301549:2022.
  • Informe conforme a la metodología del Observatorio de Accesibilidad, basada en la UNE 139803:2012.

Otras opciones de configuración:

  • Acelera la ejecución del servicio de diagnóstico al no verificar todos los enlaces existentes en la página analizada.
  • Adjunta al informe en PDF y ODS para realizar la revisón en profundidad de la muestra seleccionada.
